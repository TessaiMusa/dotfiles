# My Dotfiles

## Required packages

- BSPWM [Window manager]
- sxhkd [Shortcut key daemon]
- mpd [music player]
- mpc [client for mpd, also used in eww]
- Polybar [bar]
- eww [overlay]
- picom (ibagwan's fork) [compositor]
- zsh [shell]
  - pure [shell prompt]
  - zsh-syntax-highlighting [syntax highlighting]
  - zsh-autosuggestions [autosuggestions]
